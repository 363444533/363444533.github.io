mysql安装与基础使用

安装mysql

下载

==安装压缩包比较快，而且成功率高==点击跳转

然后进行文件解压

环境配置

打开电脑系统环境 Path 里面添加mysql安装路径

初始化mysql

在mysql根目录下创建my.ini（文件后缀为ini）

编辑文件 (注意修改basedir与datadir，这里是双斜杠)

然后，cmd窗口进入mysql目录输入mysqld --initialize对mysql进行初始化

如果mysql下面出现data文件夹则为初始化成功

进入data文件夹下

等待初始化完成会生成一个后缀名为.err的文件，里面存放的是初始化登录mysql的密码；

使用ctrl+f进行搜索password找到临时密码

删除原有服务

如果先前在电脑中装过mysql，请在cmd命令窗口输入sc delete mysql 删除之前的mysql服务。第一次安装请忽略

安装mysql服务

打开cmd窗口，输入mysqld --install 命令；（报错请使用管理员打开命令行）

登录mysql

登录命令: mysql -uroot -ppassword #password是上面初始化的err文件里的密码

修改密码

alter user 'root'@'localhost' identified with mysql_native_password by ' **这里填写新密码** ';

关闭mysql服务

Windows 键 + R 键 打开本地命令行窗口，输入services.msc快速打开电脑 “服务”，检查mysql服务，如下图所示：

将数据表里的数据导出为txt文件

进入数据库use test

导出数据

select * from shuju into outfile 't.txt'

导出的txt文件一般在mysql下面的data里面对应数据库里
